parsing do argv:
	testes válidos:
		./ircserv 4444 senha123

	inválidos:
		./ircserv -4444 senha123

		./ircserv 44o4 senha 123

		./ircserv 4444 senhçççç

		./ircserv 4444 ""
		
		./ircserv "" ""

Auth:
	-> testar qqr comando vazio

	-> testar senha incorreta

	-> testar NICK antes da senha

	-> testar dois nicks iguais




exemplo de teste:

	USER gcesar-n 0 * :Gabriel

	JOIN #sao_paulo

	PRIVMSG #sao_paulo :boa noite grupo, tudo bem?

	PRIVMSG other_user :ooi tudo bem?

	QUIT :vou sair aqui, até mais!



exemplo de comando USER:

	USER gcesar-n 0 * :Gabriel
	
	gcesar-n         username
	0                "modo", o 0 é meio q o padrão
	*                o nome do server, tbm é meio que padrão
	:Gabriel         o nome real da pessoa
