exemplo de comando USER:

	USER gcesar-n 0 * :Gabriel
	
	gcesar-n         username
	0                "modo", o 0 é meio q o padrão
	*                o nome do server, tbm é meio que padrão
	:Gabriel         o nome real da pessoa

	USER gcesar-n 0 * :Gabriel
